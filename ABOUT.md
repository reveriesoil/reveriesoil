# ReverieSoil 梦壤 · 开源说明

## 项目简介

ReverieSoil 梦壤是由微萃科技开发的 AI 视觉小说生成器。用户输入一段故事提示词，系统将自动完成故事剧本创作、角色立绘生成与语音合成，最终呈现为一部可在浏览器内直接游玩的视觉小说。

本仓库为开源版本，对外公开前端全部源代码，允许任何人免费查看、学习与个人使用。

---

## 为什么开源

我们相信好的工具应当被更多人看见和使用。开源前端代码有以下目的：

- 让开发者了解项目的技术实现，参与改进与反馈
- 让有能力自行部署的用户可以配合自己的后端服务使用
- 记录项目的成长过程，作为持续迭代的公开里程碑

## 技术栈

**前端**
- React 18 + TypeScript
- Vite 5
- framer-motion（动画）
- React Router v6

**后端（v0.2.0 发布）**
- Python 3.11 + FastAPI
- SQLite（无需额外数据库服务）
- 本地文件存储（无需对象存储）
